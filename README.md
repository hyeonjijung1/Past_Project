# 🛠️ Project 1: Subsystem C SDR Design

A hardware subsystem designed for **Software-Defined Radio (SDR)**, focusing on **Local Oscillator (LO) signal generation, UI control, and seamless system integration**.

### 📊 Design Overview:

![Block Diagram](https://github.com/hyeonjijung1/Past_Project/blob/main/Screenshot%202025-03-08%20204710.png)
*Figure 1. Subsystem C SDR Block Diagram.*

### ✨ Key Contributions:
- Designed and implemented the **Local Oscillator (LO) subsystem** to generate a precise **10MHz signal**, ensuring stable operation for SDR communication.
- Developed a **user-friendly interface (UI)** with **push-button controls** and an **LCD display**, allowing users to adjust frequency and toggle TX/RX modes.
- Wrote a detailed **technical report** outlining the subsystem’s **architecture, component selection, I/O signals, and testing methodologies**.
- Delivered both **technical presentations** and **general audience presentations**, explaining SDR principles and the significance of Subsystem C.
- Designed a **structured testing procedure** to validate the subsystem’s performance, including LO accuracy, TX/RX switching, and UI functionality.
- Collaborated with teammates to integrate Subsystem C with the SDR framework, ensuring compliance with **ICD specifications** and seamless system operation.

# 🛠️ Project 2: Voltage Sensing Board

A custom **voltage sensing PCB** designed for real-time monitoring, featuring an **STM32 microcontroller**, **IMU**, and **LCD panel** with SPI and I2C communication.

### 📜 PCB Design:

![Voltage Sensing PCB](https://github.com/hyeonjijung1/Past_Project/blob/main/Screenshot%202025-03-03%20201422.png)

### ✨ Key Contributions:
- **Designed the Altium schematic and PCB layout** for a voltage sensing board featuring an **STM32 microcontroller**, interfacing with an **IMU and LCD panel** via **SPI and I2C**.
- **Selected components** from DigiKey and implemented a **12V to 3.3V LDO** for efficient power regulation.
- **Ensured signal integrity, proper grounding, and decoupling capacitors** to enhance system stability and reliability.
